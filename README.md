# ADN Informatique | IT Monitor 🚀

**IT Monitor** est un outil de recherche de terminologie informatique conçu pour fournir des définitions techniques instantanées. Il permet aux utilisateurs de naviguer facilement dans le jargon complexe du monde de l'informatique.

## ✨ Fonctionnalités

- **Recherche Instantanée** : Trouvez des définitions pour des termes comme "Noyau Linux", "Algorithme", "SSH", etc.
- **Interface Moderne** : Design épuré avec un indicateur de statut du système en temps réel.
- **Historique de Recherche** : Gardez une trace de vos analyses précédentes.
- **Responsive Design** : Compatible avec tous les types d'écrans.

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique du site.
- **CSS3** : Mise en page avec Flexbox et polices Google (Inter).
- **JavaScript (ES6+)** : Logique de recherche et gestion de l'historique.
